# 🌙 天外家园 · 月球生态沙盒（v6）

一款基于 Streamlit 的硬核闭环生态模拟游戏。玩家在月球极区建立一座生命保障基地，通过气-水-碳-肥-电的多回路耦合，配合突发事件、科研课题、天赋分工与医疗/受伤系统，把生物圈维持得越久越好。

> 单文件运行：`streamlit run game.py`
> 依赖：`streamlit`、`pandas`、`numpy`、`matplotlib`

---

## 1. 任务初始化（开局锁定）

进入游戏先做一次性的开局战略决策（确认后全程不可改）：

- **初始高压氧气瓶**（10 kg/瓶，着陆即可用，无法补充）
- **初始乘组规模**（1~10 人）
- **初始舱室**：居住舱 / 种植舱 / 堆肥舱
- **1 号成员——选择部长**（拥有顶级天赋）：
  - **Annie 部长** —— 顶级建造（建造速度 ×3）
  - **黄部长** —— 顶级科研（科研速度 ×3）
  - **罗部长** —— 顶级探索（探索收获 ×3）
  - **郭部长** —— 顶级娱乐（每天全员 +2% 心情）
- 其余成员姓名 + 天赋随机生成（中英混合姓名池，避开职业名/政治人物）。

## 3. 生命保障系统（闭环物理）

实时演算的核心耦合方程：

- **大气**：O₂ / CO₂ 体积百分比由乘员代谢、微藻光合、作物呼吸、焚化、Sabatier、突发泄漏共同决定。
- **CDRA 智能调压**：O₂ > 24% 自动充入高压氧罐；< 19.5% 自动释放；CO₂ > 0.3% 自动洗涤入高压碳源罐。
- **水循环**：净水 ↔ 废水 ↔ 结构水。
  - 来源：微藻净化、作物蒸腾冷凝、温室冷凝、Sabatier、出舱采冰
  - 消耗：乘员代谢、模块维护、灌溉、扩建注水
  - 每日记录 `净水产量 / 净水消耗 / 净水净流量`
- **代谢常量**：O₂ 0.84、CO₂ 1.0、水 14.5、固废 0.2、食物 0.25（kg / 人 / 天）
- **堆肥**：固废 + 废水 → 肥料 + CO₂；产能受 `compost` 加成科技影响。
- **微藻**：吸 CO₂ 产 O₂、净化废水；Logistic 增长，受光照与营养限制。
- **温室**：耗肥/废水/CO₂ → 产 O₂ + 净水 + 显著心情加成。

## 4. 电力系统

- **太阳能**：`坑外光伏面积 × 0.066 kWh/m²/h × 日照时长 × 效率 mult`
- **蓄电池**：默认容量 200 kWh，独立于光伏面积。
- **焚化炉**：燃烧固废 → 增 CO₂ + 释放热值充电（0.4 kWh/kg）。
- **缺电后果**：作物 LED 补光减半、实验室停摆、居住失温扣 hull、出舱采集打折。

## 5. 农作物（批次播种）

| 作物 | 周期 | 产出类型 |
| --- | --- | --- |
| 🌾 矮秆小麦 | 65 天 | 食物 |
| 🥬 气雾生菜 | 33 天 | 食物 |
| 🥔 微型土豆 | 80 天 | 食物 |

- 周期到期按健康度比例结算收获。

| ☀️ 太阳能阵列升级 | 45 | 太阳能效率 +30% |
| 💧 闭环水回收升级 | 40 | 净水回收 +20%、维护水耗 −30% |
| 💊 辐射医学/乘组健康协议 | 40 | SPE 心情冲击减半 |

## 7. 🆕 v6 天赋系统（替代职业）

每名角色拥有 **0~2 个天赋**（概率 30% / 50% / 20%）。

**天赋类型**：
- 工作类：🔬 科研 / ⛏️ 探索 / 🔧 修理 / 🏗️ 建造 —— 放大对应工作产出
- 心理类：🎭 娱乐 —— 每天给全员加心情

**等级与倍率**：

| 等级 | 概率（普通人） | 工作倍率 | 娱乐心情/人/天 |
| --- | --- | --- | --- |
| 顶级 | 仅四部长 | ×3.0 | +2.0% |
| 高级 | 20% | ×2.0~2.9 | +1.7~1.9% |
| 中级 | 35% | ×1.5~1.9 | +1.3~1.5% |
| 低级 | 45% | ×1.0~1.4 | +1.0~1.2% |

团队倍率 = 各成员天赋倍率的平均值，再乘到团队产出上，与边际递减表叠加。

| 工作 | 说明 | 心情Δ | 健康Δ |

## 9. 🆕 v6 医疗 / 受伤 / 药物生产链

**体质**：每人初始 0.7~1.0，体质低更易生病、更难自愈（部长体质 1.0）。

**疾病系统**：
- 每日发病基础 3%，受体质与心情调节。
- 初期阶段：每日有概率自愈（体质 + 心情决定）。
- 连续 3 天未愈 → 转「加重」，每天需 1 单位药物（60% 好转），无药则每天扣 4 健康。

**受伤系统**：
- 仅探索触发，每人每日 8%。
- 60% 轻伤 / 40% 重伤。
- 轻伤每日 40% 基线自愈率；重伤需吃药（50% 好转 → 轻伤）。

**药物生产链**：
```
药用植物 🌿 → 药材 → 制药课题（实验室） → 药物 💊
                                       ↑ 治疗疾病加重 / 重伤恢复
```

## 10. 🆕 v6 招募新成员

侧边栏「🤝 招募一名新成员」：
- 居住容量：每个居住舱 / 温室容纳 2 人。
- 食物 > 5 kg 且净水 > 10 kg 才能接纳。
- 新成员姓名 + 天赋（0~2 个）随机生成；体质 0.7~1.0。

## 11. 🆕 v6 永久移除机制

- **健康 → 0**：💀 死亡 → 从乘组名册永久移除
- **心情 → 0**：🏃 叛逃 → 从乘组名册永久移除

移除时，该成员在所有实验室的派遣被解除并按姓名重映射其它成员的索引。

---

## 12. 突发事件系统

三档时间尺度，按权重抽签触发：

| 类别 | 例 | 应对模式 |
| --- | --- | --- |
| ⚡ 秒-分级 instant | 微流星撞击 / SPE 辐射暴 / 主电源中断 | 自动反射 |
| 🔧 分-小时级 ongoing | 热控泄漏 / ECLSS 故障 / 月面尘暴 | 持续 4~7 天 |
| 🫧 天-周月级 creeping | 密封微泄漏 / 杂菌膜暴发 / 结构脆化 | 15~30 天慢漂移 |

触发概率可在侧边栏调整：关闭 / 低 4% / 中 10% / 高 20%。

---

## 13. 战略物资与防御

- **月壤护甲 (m²)**：通过出舱采集获得；可被「修理」工作消耗以补 hull；同时减免辐射健康损失与心情冲击。
- **壳体完整性 σ**：撞击/脆化/缺电失温都会扣，归零 → 舱体解体失压。
- **高压氧罐 / 高压碳源罐**：CDRA 自动充放气；初始氧罐容量为开局战略决策。

---

## 14. 全局崩溃判定

任意一条触发即结束生物圈：

- CO₂ > 3.0% → 毒性崩溃
- O₂ < 18.0% → 缺氧崩溃
- 净水 < 0 / 食物 < 0
- 壳体 σ ≤ 0
- 乘组全员死亡或叛逃

游戏结束后可点击「🔄 初始化新基地」重开。
